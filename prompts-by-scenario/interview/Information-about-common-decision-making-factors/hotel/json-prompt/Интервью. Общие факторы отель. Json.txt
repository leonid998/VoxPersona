# Задача извлечения количественных характеристик из двух отчетов


Проанализируй два предоставленных отчета:
1. Первый отчет содержит факторы выбора отеля, упомянутые клиентом (с соответствующими цитатами)
2. Второй отчет представляет собой "СПИСОК НЕИЗУЧЕННЫХ ФАКТОРОВ" - факторы, которые не были упомянуты клиентом


Твоя задача - подсчитать и извлечь следующие количественные характеристики:
1. Количество выявленных (упомянутых) факторов по каждой категории
2. Количество неизученных (неупомянутых) факторов по каждой категории


Правила подсчета:
1. В отчетах факторы разделены на основные категории:
   - ЛОКАЦИОННЫЕ ФАКТОРЫ
   - ДИЗАЙН И АРХИТЕКТУРА (или ФАКТОРЫ ДИЗАЙНА И АРХИТЕКТУРЫ)
   - СЕРВИС (или ФАКТОРЫ СЕРВИСА)
   - ТЕХНОЛОГИЧЕСКИЕ ФАКТОРЫ
   - БЕЗОПАСНОСТЬ (или ФАКТОРЫ БЕЗОПАСНОСТИ)
   - МАРКЕТИНГ И КОММУНИКАЦИИ (или МАРКЕТИНГОВЫЕ ФАКТОРЫ)
   - ЭКОНОМИЧЕСКИЕ ФАКТОРЫ


2. Для упомянутых факторов:
   - Учитывай только факторы с цитатами клиента из первого отчета
   - Подсчитай количество факторов для каждой категории и подкатегории
   - В категории ДИЗАЙН И АРХИТЕКТУРА учитывай факторы из всех подкатегорий
   - В категории СЕРВИС учитывай факторы из всех подкатегорий


3. Для неизученных факторов:
   - Подсчитай количество всех неупомянутых факторов во втором отчете
   - Учитывай все факторы из "СПИСКА НЕИЗУЧЕННЫХ ФАКТОРОВ"
   - Если в списке для какой-то подкатегории указано "[Все факторы в данной подкатегории были упомянуты клиентом]", считай, что в этой подкатегории 0 неизученных факторов


4. Подсчитай общее число упомянутых и неупомянутых факторов по всем категориям.


Верни результат строго в следующем формате JSON:
Json
{
  "упомянутые_факторы": {
    "локационные_факторы": число,
    "дизайн_и_архитектура": число,
    "сервис": число,
    "технологические_факторы": число,
    "безопасность": число,
    "маркетинг_и_коммуникации": число,
    "экономические_факторы": число,
    "прочие_факторы": число,
    "общее_количество": число
  },
  "неупомянутые_факторы": {
    "локационные_факторы": число,
    "дизайн_и_архитектура": число,
    "сервис": число,
    "технологические_факторы": число,
    "безопасность": число,
    "маркетинг_и_коммуникации": число,
    "экономические_факторы": число,
    "общее_количество": число
  }
}


Где каждое "число" - это подсчитанное количество факторов в соответствующей категории (целое число). Выведите данные в формате JSON без комментариев и пояснений.
