# Задача извлечения количественных характеристик из отчета о ресторане


Проанализируй предоставленный отчет, который уже содержит выделенные факторы, относящиеся ИСКЛЮЧИТЕЛЬНО к конкретному ресторану, в котором проходит интервью.


Твоя задача - подсчитать количество этих факторов по каждой основной категории классификации.


Правила подсчета:
1. Учитывай все факторы, представленные в отчете (они уже отфильтрованы и относятся только к текущему ресторану).


2. В отчете факторы разделены на следующие основные категории:
   - ЛОКАЦИЯ И ЭКСТЕРЬЕР
   - ПЕРВОЕ ВПЕЧАТЛЕНИЕ
   - ДИЗАЙН И ИНТЕРЬЕР
   - КУХНЯ И НАПИТКИ
   - СЕРВИС
   - ДОПОЛНИТЕЛЬНЫЕ УСЛУГИ
   - БЕЗОПАСНОСТЬ И ГИГИЕНА
   - МАРКЕТИНГ И КОММУНИКАЦИИ
   - ЭКОНОМИЧЕСКИЕ ФАКТОРЫ
   - ПРОЧИЕ ФАКТОРЫ


3. Подсчитывай общее количество факторов по каждой основной категории, не разбивая на подкатегории.


4. Учитывай только уникальные факторы, каждый из которых обычно подкреплен цитатой клиента.


5. Подсчитай общее количество всех факторов по всем категориям.


Верни результат строго в следующем формате JSON:
Json
{
  "локация_и_экстерьер": число,
  "первое_впечатление": число,
  "дизайн_и_интерьер": число,
  "кухня_и_напитки": число,
  "сервис": число,
  "дополнительные_услуги": число,
  "безопасность_и_гигиена": число,
  "маркетинг_и_коммуникации": число,
  "экономические_факторы": число,
  "прочие_факторы": число,
  "общее_количество_факторов": число
}


Где каждое "число" - это подсчитанное количество факторов в соответствующей категории (целое число). Выведите данные в формате JSON без комментариев и пояснений.
