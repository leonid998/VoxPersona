Проанализируй предоставленный отчет, который уже содержит выделенные факторы, относящиеся ИСКЛЮЧИТЕЛЬНО к конкретному отелю, в котором проходит интервью.


Твоя задача - подсчитать количество этих факторов по каждой основной категории классификации.


Правила подсчета:
1. Учитывай все факторы, представленные в отчете (они уже отфильтрованы и относятся только к текущему отелю).


2. В отчете факторы разделены на следующие основные категории:
   - ЛОКАЦИОННЫЕ ФАКТОРЫ
   - ФАКТОРЫ ДИЗАЙНА И АРХИТЕКТУРЫ (включая все подкатегории)
   - ФАКТОРЫ СЕРВИСА (включая все подкатегории)
   - ТЕХНОЛОГИЧЕСКИЕ ФАКТОРЫ
   - ФАКТОРЫ БЕЗОПАСНОСТИ
   - МАРКЕТИНГОВЫЕ ФАКТОРЫ
   - ПРОЧИЕ ФАКТОРЫ


3. Для категории ФАКТОРЫ ДИЗАЙНА И АРХИТЕКТУРЫ учитывай все факторы из всех подкатегорий и суммируй их в общее число для этой категории.


4. Для категории ФАКТОРЫ СЕРВИСА учитывай все факторы из всех подкатегорий и суммируй их в общее число для этой категории.


5. Подсчитай общее количество всех факторов по всем категориям.


6. Учитывай только уникальные факторы, каждый из которых обычно подкреплен цитатой клиента.


Верни результат строго в следующем формате JSON:
Json
{
  "локационные_факторы": число,
  "факторы_дизайна_и_архитектуры": число,
  "факторы_сервиса": число,
  "технологические_факторы": число,
  "факторы_безопасности": число,
  "маркетинговые_факторы": число,
  "прочие_факторы": число,
  "общее_количество_факторов": число
}


Где каждое "число" - это подсчитанное количество факторов в соответствующей категории (целое число). Выведите данные в формате JSON без комментариев и пояснений.
