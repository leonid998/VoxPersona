Проанализируй предоставленный отчет о качестве проведения глубинного интервью и извлеки из него значение "Итоговая оценка качества интервью" (Z%). 


Правила извлечения:
1. Найди в тексте отчета раздел "Общее заключение".
2. В этом разделе найди строку, содержащую фразу "Итоговая оценка качества интервью: Z%" или аналогичную формулировку с числовым значением.
3. Извлеки числовое значение (только число без знака процента).
4. Если оценка представлена в другом формате или с другой формулировкой, определи числовое значение, которое представляет итоговую оценку качества интервью.


Верни результат строго в следующем формате JSON:
Json
{
  "итоговая_оценка_качества_интервью": число
}


Где "число" - это извлеченное числовое значение (целое число или число с плавающей точкой). Выведите данные в формате JSON без комментариев и пояснений.
