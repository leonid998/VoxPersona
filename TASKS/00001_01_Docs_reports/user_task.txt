При нажатии на кнопку Мои отчеты должно происходить следующее:
Текущие Отчеты и текущее меню с выбором отчета не присылать (текущая реализация не подходит, нужна другая, текущую
реализацию не использовать)
Нужна другая реализация6
Нужно чтобы в ответ присылался txt файл с полным списком отчетов, пронумерованный, чтобы дальше по этому номеру
можно было обращаться к самому отчету. Внизу меню с кнопками выбора. Кнопка Посмотреть - Возвращает файл самого
отчета в txt по указанному номеру
Переименовать - Меняет наименование отчета с системного на выбраный по указанному номеру
Удалить - удаляет отчет из системы хранения по указанному номеру.
По нажатию на любую кнопку бот предлагает указать Номер отчета для дальнейших действий, а потом выполняет
взависимости от нажатой кнопки. Перед Действием нужно подтверждение пользователя.
Используй существующий механизм очистки меню и артефактов после использования.
